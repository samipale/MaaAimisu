# MaaAimisu

</div>

本项目为基于 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 的あいりすミスティリア日常助手。

目前仍在开发阶段。

## 常见问题
* 如使用DMM端且游戏分辨率不是1280*720，需要自行在根目录的interface.json中删除display_raw一行。
* 如有其他问题，请提交Issue。

## 参与开发

请阅读 [如何开发](https://maafw.com/docs/1.1-QuickStarted)。
* 本项目基于方案二，即以JSON为主，复杂逻辑使用agent实现。
* 向本仓库提交改动前，请参考模板仓库的 [PR 规范](https://github.com/MaaXYZ/MaaPracticeBoilerplate/blob/main/docs/zh_cn/develop/pull_request_guidelines.md)。
